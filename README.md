# 🚗 Car Trades – Online Car Marketplace

## 📖 Project Overview
Car Trades is a full-stack ecommerce web application developed to simplify buying and selling second-hand cars in Gandhinagar. The platform focuses on verified listings, secure transactions, and improved trust between buyers and sellers.

---

## 🎯 Objective
- Provide a dedicated platform for used car trading in Gandhinagar
- Enable secure and verified transactions
- Offer car valuation and document verification
- Improve trust compared to generic platforms like OLX

---

## 🚀 Key Features

### 👤 User Features
- User Registration & Login
- Browse & Search Cars
- Add/Edit/Delete Listings
- Buy Car Functionality
- Secure Payment (User → Admin)
- Test Drive Scheduling
- Document Verification Requests

### 🛠 Admin Features
- Admin Dashboard
- Manage Users
- Approve Car Listings
- Verify Documents
- Handle Test Drive Requests
- Payment Management (Admin → Seller)

---

## 🏗 System Architecture
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Payment Gateway: Razorpay

---

## 🔐 System Validation
- Email & Phone Verification
- Password Encryption
- Secure Payment Integration

## 📊 Database
The project includes structured tables for:
